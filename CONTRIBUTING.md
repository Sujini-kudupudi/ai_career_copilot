# Contributing

Thank you for improving AI Career Copilot.

## Local Development

1. Create and activate a virtual environment.
2. Install dependencies with `pip install -r requirements.txt`.
3. Run tests with `python -m pytest`.
4. Run the app with `streamlit run app/streamlit_app.py`.

## Coding Guidelines

- Keep analysis logic inside `src/career_copilot`.
- Keep UI logic inside `app`.
- Add or update tests when changing scoring behavior.
- Avoid committing resumes, personal data, or uploaded documents.
